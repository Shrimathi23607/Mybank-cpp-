# MyBank - Console Based Banking System

A menu-driven banking system built in C++.

## Features
- Create bank account
- Deposit and Withdraw money
- Check balance
- View transaction history with date and time

## Concepts Used
- Object Oriented Programming (OOP)
- Classes and Objects
- Vectors
- Constructors

## How to Run
- Compile: g++ MyBank.cpp -o MyBank
- Run: ./MyBank
